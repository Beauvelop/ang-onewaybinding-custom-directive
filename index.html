Function One-Way Binding Example

    
  
  • Step I. in index.html..
    • 

    < info-dir> < / info-dir>
    

  • Step II. create file
    • 

    infoDir.js
    

  • Step III. add a var on scope in mainCtrl.js
    • 

    $scope.name = "Leroy MmmJenkins";
    

  • Step IV. add { {  }} to be interpolated
    • 

    { { name } }  (in infoDir.html)
    

    Thus far...{ { name } } WILL show as "Leroy"..!

    It IS on scope (in the controller)

    Because NO Scope has been explicitly declared in the directive!
    *There is no scope object in the directive!
    
IT WILL TAKE THE SAME SCOPE AS IT'S PARENT!
    


  • Step V. add a scope var to the main controllerin myCtrl.js
    • 

       .controller("myCtrl", function($scope) {
        $scope.name = "Leroy MmmJenkins";
  }) 
    
  
  • Step VI. add a scope object to this directive--and, thus, giving it itsown isolated scopein myCtrl.js
    • 
       templateUrl: "./infoDir.html",
             scope: {},
        }
 
    

  • Step VII. Now, you can give it itsown controller & value of a 'name' variable!in myCtrl.js
    • 
      templateUrl: "./infoDir.html",
            scope: {},
       controller: function($scope) {
   $scope.name = 'Bill Smith'; // this $scope value will win over it's parent
 }
